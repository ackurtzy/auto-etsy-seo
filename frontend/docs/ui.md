Experiments page - 5 tabs/modes within. Next to where to switch to each tab, it shows how many things are in that tab
For all tabs, the information is in a single column where each row is a listing with the listing preview + the other tab-specific information. You can sort by date in some form (depending on the specifics of the tab. For proposals, it could be by proposal date. For active, it could be by end date, etc). You can also search by title
When you click to make the item go from one tab to the next, it immediately disappears from this tab, and appears on the next tab (or is shown as generating in the case of inactive goes to generating)

Inactive (you can come up with better name)- Active listings with no proposals or active tests
Show listing preview where each listing is a checkbox. At the top, there is a generate proposals button that does it with the selected listings. Once you click this. The listings disappear from this page and are moved to the proposals page.
On this page there is a also a small place to control settings, such as set how long the experiment should run for, which model should be used to run the experiments, etc
Proposals
Shows listing preview. You click on the listing row accordion style for the 3 different experiment options to appear. You can select one and apply it with a button that is also in the accordion. You can also regenerate the proposal
At the top, there is an indicator of how many experiments are currently being generated. As they are finished being generated, they appear on this page

Active - Review and end current experiments
Shows listing preview, experiment details, and performance
There is an end experiment early button. If you click it, then you are prompted to either go back and let the experiment continue, keep the change, or revert the change. To make this change, you are given the performance details and reminded of the details of the change.

Finished (waiting to be resolved by the user):
Shows listing preview and Gives summary of listing experiment. There are a few options that appear accordion style: keep the change, revert the change, or run for another x amount of time. To make this choice, you are given the performance details and reminded of the details of the change

Completed - Shows listing preview. Whether the experiment was kept or reverted. The performance impact

Listings Page - Catalog of all the listings with quick facts about number of experiments run on it, lifetime performance increase from those experiments.
If you click on the listing details page, it brings you to an all page that shows all current listing details. You can scroll through the images, see all past experiments and proposals, etc
